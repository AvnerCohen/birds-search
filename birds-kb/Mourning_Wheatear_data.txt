<p>The <b>mourning wheatear</b> (<i>Oenanthe lugens</i>) is a <a href="/wiki/Bird" title="Bird">bird</a>, one of 14 species of wheatear found in northern <a href="/wiki/Africa" title="Africa">Africa</a> and the <a href="/wiki/Middle_East" title="Middle East">Middle East</a>. It is a small passerine in a group formerly classed as members of the thrush family Turdidae, but now more generally considered to be part of the Old World flycatcher family <a href="/wiki/Muscicapidae" class="mw-redirect" title="Muscicapidae">Muscicapidae</a>.
</p><p>The mourning wheatear was first described by <a href="/wiki/Martin_Lichtenstein" class="mw-redirect" title="Martin Lichtenstein">Martin Lichtenstein</a> in 1823. It is found in semi-desert areas in North Africa and the Middle East. It is sexually dimorphic with the females sporting more subtle plumage.
</p><p>An intriguing dark <a href="/wiki/Polymorphism_(biology)" title="Polymorphism (biology)">morph</a> of the mourning wheatear (the so-called basalt wheatear) occurs in the basalt desert of northeast Jordan.
</p><p>The north African subspecies <i>halophila</i>, considered by some to be a separate species, western mourning wheatear, occurs from Morocco east to western Egypt. The Egyptian populations are discussed in Baha El Din and Baha El Din (2000).<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup> These birds differ from typical <i>halophila</i> in exhibiting less sexual dimorphism, and displaying a prominent white wingbar, and thus are closer to the nominate race.
</p><p>Panov (Wheatears of Palearctic, 2005) discusses the latest taxonomy for the mourning wheatear superspecies. The mourning wheatear is split from <a href="/wiki/Abyssinian_wheatear" title="Abyssinian wheatear">Abyssinian wheatear</a> (<i>Oenanthe lugubris</i>), which is the species found south of the Sahara.
</p><p>It has been recorded in the following countries: <a href="/wiki/Algeria" title="Algeria">Algeria</a>, <a href="/wiki/Bahrain" title="Bahrain">Bahrain</a>, <a href="/wiki/Cyprus" title="Cyprus">Cyprus</a>, <a href="/wiki/Egypt" title="Egypt">Egypt</a>, <a href="/wiki/Iran" title="Iran">Iran</a>, <a href="/wiki/Iraq" title="Iraq">Iraq</a>, <a href="/wiki/Israel" title="Israel">Israel</a>, <a href="/wiki/Jordan" title="Jordan">Jordan</a>, <a href="/wiki/Kuwait" title="Kuwait">Kuwait</a>, <a href="/wiki/Lebanon" title="Lebanon">Lebanon</a>, <a href="/wiki/Libya" title="Libya">Libya</a>, <a href="/wiki/Morocco" title="Morocco">Morocco</a>, <a href="/wiki/Oman" title="Oman">Oman</a>, <a href="/wiki/Pakistan" title="Pakistan">Pakistan</a>, <a href="/wiki/Qatar" title="Qatar">Qatar</a>, <a href="/wiki/Saudi_Arabia" title="Saudi Arabia">Saudi Arabia</a>, <a href="/wiki/Syria" title="Syria">Syria</a>, <a href="/wiki/Tunisia" title="Tunisia">Tunisia</a>, <a href="/wiki/Turkey" title="Turkey">Turkey</a>, the <a href="/wiki/United_Arab_Emirates" title="United Arab Emirates">United Arab Emirates</a>, and <a href="/wiki/Yemen" title="Yemen">Yemen</a>.
</p>
