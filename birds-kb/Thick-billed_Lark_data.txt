<p>The <b>Thick-billed Lark</b> (<i>Rhamphocoris clotbey</i>) is a species of <a href="/wiki/Lark" title="Lark">lark</a> in the <a href="/wiki/Alaudidae" title="Alaudidae" class="mw-redirect">Alaudidae</a> family. It is <a href="/wiki/Monotypic" title="Monotypic" class="mw-redirect">monotypic</a> within the genus <i><b>Rhamphocoris</b></i>.<sup class="Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources from October 2011">citation needed</span></a></i>]</sup> It is found in <a href="/wiki/Algeria" title="Algeria">Algeria</a>, <a href="/wiki/Egypt" title="Egypt">Egypt</a>, <a href="/wiki/Israel" title="Israel">Israel</a>, <a href="/wiki/Jordan" title="Jordan">Jordan</a>, <a href="/wiki/Kuwait" title="Kuwait">Kuwait</a>, <a href="/wiki/Libya" title="Libya">Libya</a>, <a href="/wiki/Mauritania" title="Mauritania">Mauritania</a>, <a href="/wiki/Morocco" title="Morocco">Morocco</a>, <a href="/wiki/Saudi_Arabia" title="Saudi Arabia">Saudi Arabia</a>, <a href="/wiki/Tunisia" title="Tunisia">Tunisia</a>, and <a href="/wiki/Yemen" title="Yemen">Yemen</a>.</p>
<p>Its natural <a href="/wiki/Habitat" title="Habitat">habitats</a> are <a href="/wiki/Subtropical" title="Subtropical" class="mw-redirect">subtropical</a> or <a href="/wiki/Tropical" title="Tropical" class="mw-redirect">tropical</a> dry <a href="/wiki/Shrubland" title="Shrubland">shrubland</a> and hot <a href="/wiki/Desert" title="Desert">deserts</a>.</p>
