<p>The <b>Crowned Sandgrouse</b> (<i>Pterocles coronatus</i>) is a species of <a href="/wiki/Bird" title="Bird">bird</a> in the <a href="/wiki/Pteroclididae" title="Pteroclididae" class="mw-redirect">Pteroclididae</a> family. It is found in <a href="/wiki/Afghanistan" title="Afghanistan">Afghanistan</a>, <a href="/wiki/Algeria" title="Algeria">Algeria</a>, <a href="/wiki/Chad" title="Chad">Chad</a>, <a href="/wiki/Egypt" title="Egypt">Egypt</a>, <a href="/wiki/India" title="India">India</a>, <a href="/wiki/Iran" title="Iran">Iran</a>, <a href="/wiki/Israel" title="Israel">Israel</a>, <a href="/wiki/Jordan" title="Jordan">Jordan</a>, <a href="/wiki/Libya" title="Libya">Libya</a>, <a href="/wiki/Mali" title="Mali">Mali</a>, <a href="/wiki/Mauritania" title="Mauritania">Mauritania</a>, <a href="/wiki/Morocco" title="Morocco">Morocco</a>, <a href="/wiki/Niger" title="Niger">Niger</a>, <a href="/wiki/Oman" title="Oman">Oman</a>, <a href="/wiki/Pakistan" title="Pakistan">Pakistan</a>, <a href="/wiki/Saudi_Arabia" title="Saudi Arabia">Saudi Arabia</a>, <a href="/wiki/Sudan" title="Sudan">Sudan</a>, <a href="/wiki/Tunisia" title="Tunisia">Tunisia</a>, <a href="/wiki/United_Arab_Emirates" title="United Arab Emirates">United Arab Emirates</a>, <a href="/wiki/Western_Sahara" title="Western Sahara">Western Sahara</a>, and <a href="/wiki/Yemen" title="Yemen">Yemen</a>.</p>
<h2><span class="editsection">[<a href="/w/index.php?title=Crowned_Sandgrouse&amp;action=edit&amp;section=1" title="Edit section: Subspecies">edit</a>]</span> <span class="mw-headline" id="Subspecies">Subspecies</span></h2>
<p>There are five subspecies:<sup id="cite_ref-1" class="reference"><a href="#cite_note-1"><span>[</span>2<span>]</span></a></sup></p>
<ul>
<li><i>P. c. coronatus</i> - Sahara, Morocco to Red Sea</li>
<li><i>P. c. vastitas</i> - Sinai, Israel and Jordan</li>
<li><i>P. c. saturatus</i> - Oman</li>
<li><i>P. c. atratus</i> - Saudi Arabia, Iran and Afghanistan</li>
<li><i>P. c. ladas</i> - Pakistan</li>
</ul>
