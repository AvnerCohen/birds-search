<p>The <b>Ethiopian Swallow</b> (<i>Hirundo aethiopica</i>) is a species of <a href="/wiki/Bird" title="Bird">bird</a> in the <a href="/wiki/Hirundinidae" title="Hirundinidae" class="mw-redirect">Hirundinidae</a> family. It is found in <a href="/wiki/Benin" title="Benin">Benin</a>, <a href="/wiki/Burkina_Faso" title="Burkina Faso">Burkina Faso</a>, <a href="/wiki/Cameroon" title="Cameroon">Cameroon</a>, <a href="/wiki/Central_African_Republic" title="Central African Republic">Central African Republic</a>, <a href="/wiki/Chad" title="Chad">Chad</a>, <a href="/wiki/Democratic_Republic_of_the_Congo" title="Democratic Republic of the Congo">Democratic Republic of the Congo</a>, <a href="/wiki/Ivory_Coast" title="Ivory Coast">Ivory Coast</a>, <a href="/wiki/Eritrea" title="Eritrea">Eritrea</a>, <a href="/wiki/Ethiopia" title="Ethiopia">Ethiopia</a>, <a href="/wiki/Ghana" title="Ghana">Ghana</a>, <a href="/wiki/Guinea" title="Guinea">Guinea</a>, <a href="/wiki/Israel" title="Israel">Israel</a>, <a href="/wiki/Kenya" title="Kenya">Kenya</a>, <a href="/wiki/Mali" title="Mali">Mali</a>, <a href="/wiki/Niger" title="Niger">Niger</a>, <a href="/wiki/Nigeria" title="Nigeria">Nigeria</a>, <a href="/wiki/Senegal" title="Senegal">Senegal</a>, <a href="/wiki/Somalia" title="Somalia">Somalia</a>, <a href="/wiki/Sudan" title="Sudan">Sudan</a>, <a href="/wiki/Tanzania" title="Tanzania">Tanzania</a>, <a href="/wiki/Togo" title="Togo">Togo</a>, and <a href="/wiki/Uganda" title="Uganda">Uganda</a>.</p>
