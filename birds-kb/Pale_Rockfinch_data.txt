<p>The <b>Pale Rockfinch</b> or <b>Pale Rock Sparrow</b> (<i>Carpospiza brachydactyla</i>) is a small <a href="/wiki/Sparrow" title="Sparrow">sparrow</a> found in the <a href="/wiki/Middle_East" title="Middle East">Middle East</a> and <a href="/wiki/Central_Asia" title="Central Asia">Central Asia</a>. It is the only member of the genus <i><b>Carpospiza</b></i>. Some authorities include it in the genus <i><a href="/wiki/Petronia" title="Petronia">Petronia</a></i>.<sup id="cite_ref-HBW_1-0" class="reference"><a href="#cite_note-HBW-1"><span>[</span>2<span>]</span></a></sup> Others have placed it in the <a href="/wiki/Finch" title="Finch">finch</a> family due to some similarities in behaviour and form, but the anatomy of its tongue exhibits characteristic sparrow features.<sup id="cite_ref-HBW_1-1" class="reference"><a href="#cite_note-HBW-1"><span>[</span>2<span>]</span></a></sup><sup id="cite_ref-2" class="reference"><a href="#cite_note-2"><span>[</span>3<span>]</span></a></sup> The countries in which it occurs are <a href="/wiki/Afghanistan" title="Afghanistan">Afghanistan</a>, <a href="/wiki/Armenia" title="Armenia">Armenia</a>, <a href="/wiki/Azerbaijan" title="Azerbaijan">Azerbaijan</a>, <a href="/wiki/Cyprus" title="Cyprus">Cyprus</a>, <a href="/wiki/Djibouti" title="Djibouti">Djibouti</a>, <a href="/wiki/Egypt" title="Egypt">Egypt</a>, <a href="/wiki/Eritrea" title="Eritrea">Eritrea</a>, <a href="/wiki/Ethiopia" title="Ethiopia">Ethiopia</a>, <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a>, <a href="/wiki/Iran" title="Iran">Iran</a>, <a href="/wiki/Iraq" title="Iraq">Iraq</a>, <a href="/wiki/Israel" title="Israel">Israel</a>, <a href="/wiki/Jordan" title="Jordan">Jordan</a>, <a href="/wiki/Kuwait" title="Kuwait">Kuwait</a>, <a href="/wiki/Lebanon" title="Lebanon">Lebanon</a>, <a href="/wiki/Oman" title="Oman">Oman</a>, <a href="/wiki/Pakistan" title="Pakistan">Pakistan</a>, <a href="/wiki/Qatar" title="Qatar">Qatar</a>, <a href="/wiki/Saudi_Arabia" title="Saudi Arabia">Saudi Arabia</a>, <a href="/wiki/Sudan" title="Sudan">Sudan</a>, <a href="/wiki/Syria" title="Syria">Syria</a>, <a href="/wiki/Turkey" title="Turkey">Turkey</a>, <a href="/wiki/Turkmenistan" title="Turkmenistan">Turkmenistan</a>, <a href="/wiki/United_Arab_Emirates" title="United Arab Emirates">United Arab Emirates</a>, and <a href="/wiki/Yemen" title="Yemen">Yemen</a>.<sup id="cite_ref-IUCN2012_0-1" class="reference"><a href="#cite_note-IUCN2012-0"><span>[</span>1<span>]</span></a></sup></p>
<p>Its natural <a href="/wiki/Habitat" title="Habitat">habitats</a> are <a href="/wiki/Subtropical" title="Subtropical" class="mw-redirect">subtropical</a> or <a href="/wiki/Tropical" title="Tropical" class="mw-redirect">tropical</a> dry <a href="/wiki/Shrubland" title="Shrubland">shrubland</a> and temperate <a href="/wiki/Grassland" title="Grassland">grassland</a>.<sup id="cite_ref-HBW_1-2" class="reference"><a href="#cite_note-HBW-1"><span>[</span>2<span>]</span></a></sup></p>
